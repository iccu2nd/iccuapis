Sasane APIS

API pribadi buat kebutuhan bot dan project kamu sendiri. Satu base URL, banyak endpoint siap pakai — download, gambar, tools, sampai search — tinggal request lewat HTTP GET, respons langsung balik dalam JSON atau file.

Explore Docs Cara Pakai

Cara Pakai

  1. Pilih Endpoint: Buka halaman Docs, lalu pilih kategori dan endpoint yang mau dipakai — AI, Search, Image, Stalk, Downloader, atau Tools.

  2. Kirim Request: Semua endpoint pakai method GET biasa, tinggal request ke base URL + path endpoint + query parameter yang dibutuhkan.

  3. Ambil Hasil: Respons langsung balik dalam format JSON atau file (gambar/audio/video), siap dipakai langsung di project kamu. Nggak perlu API key buat mulai coba-coba.

Contoh Request

const res = await fetch("https://api.sasane.eu.cc/search/youtube?query=lofi");
const data = await res.json();
console.log(data);
import requests

res = requests.get("https://api.sasane.eu.cc/search/youtube", params={"query": "lofi"})
print(res.json())
curl "https://api.sasane.eu.cc/search/youtube?query=lofi"

Setiap endpoint punya parameter beda-beda, semuanya bisa dilihat dan dicoba langsung di halaman docs.

Kategori Endpoint

AI

Endpoint AI kayak image generation dan text processing (DeepAI), tinggal panggil langsung dari kode kamu.

Search

Cari YouTube dan lirik lagu, hasilnya balik rapi dalam JSON, gampang diolah lagi di bot atau aplikasi.

Image

Ambil gambar dari sumber seperti Pixiv, Konachan, dan Blue Archive lewat satu endpoint sederhana.

Stalk

Ambil informasi publik dari profil TikTok dengan satu request, cocok buat fitur cek akun di bot kamu.

Downloader

Download audio dari Spotify dan YouTube (ytmp3) langsung dari link, tanpa proses ribet.

Tools

Utilitas harian: remove background gambar, skip shortlink iklan, spam OTP, sampai convert web ke APK.
