Cara Penggunaan

Semua endpoint pakai method GET biasa. Tinggal request ke base URL + path endpoint, tambahkan query parameter yang dibutuhkan, lalu ambil hasilnya dari respons. Nggak perlu API key buat mulai coba-coba.

const res = await fetch("https://api.sasane.eu.cc/search/youtube?query=lofi");
const data = await res.json();
console.log(data);
import requests

res = requests.get("https://api.sasane.eu.cc/search/youtube", params={"query": "lofi"})
print(res.json())
curl "https://api.sasane.eu.cc/search/youtube?query=lofi"

Setiap endpoint punya parameter beda-beda, semuanya bisa dilihat dan dicoba langsung di halaman docs.
